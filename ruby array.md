# What is Ruby Array 
Array เป็นโครงสร้างข้อมูลชนิดหนึ่ง สามารถใช้เก็บหลายตัวแปรได้ ซึ่งข้อมูลข้างในจะเป็นประเภทเดียวกันหรือหลากหลายประเภทก็ได้ 
## Example
```ruby
b = [1,2,"A",true]
print b
```
<details> <summary>Output</summary>
[1, 2, "A", true]
</details>

# How to Creat a Ruby Array
```ruby
a = [1,2,3,"Cat"]
print a
```
<details> <summary>Output</summary>
[1, 2, 3, "Cat"]
</details>
